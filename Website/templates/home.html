Welcome to My Website!

Hello, I'm Dylan, a Cybersecurity professional working, experimenting with, and deploying technology with the hope of growing and displaying my knowledge and technical ability. This website is a simple web application written using the Flask web framework for Python, containerized using Docker, and deployed using Terraform to the cloud. The goal of creating this is to not just demonstrate technical ability, but to create a guide and make all resources available for others to do the same.

This website itself is a project to practice and demonstrate:
The Flask framework

Flask is a lightweight Python web framework that simplifies the development of web applications by providing essential tools and a flexible structure. Scripting and automation in Python is a part of my day-to-day work and life and I wanted to take on the challenge of using something familiar in a new way.

Containerization with Docker

Containerization streamlines software development and deployment by encapsulating applications and their dependencies, ensuring consistency across various environments. Docker is the most popular tool to achieve this. This web application was containerized using Docker.

Infrastructure as code with Terraform

Infrastructure as Code (IaC) enables the automated provisioning and management of IT infrastructure through code, fostering scalability, consistency, and version control in deployment processes. Terraform was used in this instance to deploy the Docker image of this web application to the cloud.
